Comments ¶

  • 短評功能：

    • 使用者可以…
      • 點擊 更多 或是 我也要說 進入短評 Modal
      • 在輸入文字框中新增自己的短評
        • 每則短評最多輸入 20 個字
        • 最多可輸入 5 則短評
      • 點選 編輯 進入短評刪除模式
        • 點選短評旁的 x 可以刪除短評，會跳出 Modal 確認是否刪除
      • 點選未高亮顯示的熱門短評
        • 會有 +1 動畫效果
        • 該短評會以高亮顯示
      • 點選已高亮顯示的熱門短評
        • 非自己建立的
          • 該短評會恢復成未高亮顯示的狀態
        • 自己建立的
          • 跳出 toast 顯示使用編輯功能來刪除短評

  • 拆分原因：

    • 短評功能區塊

get comments  ¶

comments
GET/comments{?titleId,userId}

  • 描述：列出該劇集的所有評論

  • 使用場景：用戶查看特定影片的評論

### pseudo code
def get_comments_for_title(title_id, user_id)
  all_comments = Database.sql("SELECT * FROM comments WHERE title_id = ? ORDER BY counter DESC")
  self_create_comments = Database.sql("SELECT * FROM comments WHERE title_id = ? AND user_id =?", title_id, user_id)

  {
    selfCreateComments: self_create_comments || [],
    allComments: all_comments
  }
end
